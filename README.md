# Telegram Support Bot (Personal Routing)

Асинхронный Telegram-бот для организации службы поддержки с системой персонального распределения тикетов «Клиент — Свободный оператор».

## Основные возможности

*   **Полная асинхронность**: Использование `aiogram 3.x` для работы с Telegram API и `aiosqlite` для работы с базой данных SQLite.
*   **Персональные чаты**: Бот выступает посредником, пересылая сообщения между клиентом и назначенным оператором в личных сообщениях.
*   **Система очередей**: Если все операторы заняты или не в сети, клиенты автоматически ставятся в очередь (`waiting`).
*   **Автоматическое распределение**: Как только оператор освобождается или выходит в сеть, бот автоматически назначает ему самого старого клиента из очереди.
*   **Управление статусом**: Операторы могут уходить на перерыв, приостанавливая получение новых тикетов.

## Технологический стек

*   Python 3.8+
*   aiogram 3.x (Telegram Bot Framework)
*   aiosqlite (Async SQLite wrapper)
*   python-dotenv (Управление переменными окружения)

## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url_вашего_репозитория>
   cd "Bot Panel"
   ```

2. **Установите зависимости:**
   ```bash
   pip install aiogram aiosqlite python-dotenv
   ```

3. **Настройте переменные окружения:**
   Создайте файл `.env` в корневой папке и добавьте туда ваш токен:
   ```env
   BOT_TOKEN=ваш_токен_от_botfather
   ```

4. **Запустите бота:**
   ```bash
   python bot.py
   ```

## Команды

### Для операторов:
*   `/start admin_secret` — регистрация в качестве оператора (секретное слово настраивается в коде).
*   `/status` — переключение режима работы (В сети / На перерыве).
*   `/close` — завершение текущего диалога и переход к следующему клиенту из очереди.

### Для клиентов:
*   Достаточно просто написать любое сообщение боту в ЛС, чтобы инициировать запрос в поддержку.
